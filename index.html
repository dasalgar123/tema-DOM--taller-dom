    Búsqueda en Tabla de concepto de DOM


    Palabra Definición
    DOM Modelo de Objetos del Documento; una representación del documento HTML en forma de objeto que permite su manipulación a través de JavaScript.
    Elemento Una parte individual de un documento HTML, como un <div>, <p>, o <h1>, que puede ser manipulado mediante JavaScript.
    Etiqueta Una instrucción en el código HTML que define un elemento, como <a> o <img>.
    Nodo Cualquier objeto en el DOM, incluyendo elementos, atributos y texto. Los nodos son las unidades básicas del DOM.
    Texto El contenido textual de un nodo, que puede ser accesible y modificable mediante JavaScript, utilizando propiedades como textContent.
    Selector Método para seleccionar elementos en el DOM, usado en métodos como document.querySelector() y document.getElementById().
    Evento Una acción que ocurre en el documento, como un clic o un movimiento del ratón, que se puede manejar con JavaScript.
    Escucha de eventos Función que se ejecuta cuando ocurre un evento en un elemento del DOM, usando element.addEventListener().
    Modificar Cambiar las propiedades o el contenido de un elemento del DOM usando JavaScript, como element.textContent o element.style.
    Crear Generar nuevos elementos en el DOM usando document.createElement().
    Eliminar Borrar elementos del DOM utilizando métodos como element.remove() o parentNode.removeChild().
    Clonar Crear una copia de un elemento existente en el DOM usando element.cloneNode().
    Traversing Navegar a través de los nodos del DOM utilizando propiedades como parentNode, childNodes y nextSibling.
    InnerHTML Propiedad que permite obtener o establecer el contenido HTML de un elemento, incluyendo sus etiquetas.
    Style Propiedad que permite acceder y modificar el estilo CSS de un elemento del DOM.
    JSON Formato de intercambio de datos usado comúnmente en combinación con el DOM para cargar datos en la página.
    Atributo Una propiedad de un elemento HTML que proporciona información adicional, como src en una etiqueta <img>.
